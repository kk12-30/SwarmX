## 📋 项目简介

**SwarmX** 是一款面向企业级安全评估的智能化渗透测试平台，采用 **多 Agent 协作架构**，模拟红队专家团队的协作模式，实现从信息收集、漏洞探测到利用验证的全流程自动化。

### 核心设计理念

- **蜂群智能**：多 Agent 并行协作，各专其职
- **知识沉淀**：知识图谱驱动的记忆与推理
- **人机协同**：Web 可视化界面 + AI 自主决策
- **持续进化**：支持自定义工具集成与策略扩展
